## Google Cloud Configuration

This directory is meant to hold a description of the WCFC-Apps Google Cloud
deployment.  This is programmatically generated by running:

```
gcloud asset export \
  --project=wcfc-apps \
  --content-type=resource \
  --output-path=./wcfc-apps-gcloud.ndjson
```

Then once the operation completes, run

```
gcloud storage cp gs://wcfc-apps/wcfc-apps-gcloud.ndjson .
```

to download the file locally.

Once you have the file, use jq to inspect it; for example,

```
jq 'select(.asset_type=="run.googleapis.com/Service") | {name, asset_type, update_time}' | wcfc-apps-gcloud.ndjson
```
